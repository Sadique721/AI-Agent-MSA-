🌟 What is MSA?

No cloud. No API. No tracking.

⚙️ Installation Guide
🔽 Step 1: Clone Repository
git clone <your-repo-url>
cd msa_agent
🧠 Step 2: Add AI Models

Place models inside the models/ folder:

Model Type	Path
🎤 Vosk	models/vosk/
🗣️ Whisper	models/whisper.cpp/
🤖 LLM (GGUF)	models/llm/
🛠️ Step 3: Install Dependencies
System Dependencies
bash scripts/install_deps.sh
Python Packages
pip install -r requirements.txt
🎙️ Step 4: Train Your Voice (IMPORTANT 🔐)
python scripts/train_speaker.py

👉 This ensures only you can activate MSA.

🚀 Step 5: Run MSA
bash run.sh

🎉 Now say: "Hey MSA"
